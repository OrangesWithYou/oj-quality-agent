# Code of Conduct

## Our Pledge

我们希望 质检难题智能体_鲍元吉_20260617 的讨论和协作保持专业、友善、务实。参与者应尊重不同背景、经验水平和观点。

## Expected Behavior

- 聚焦问题本身，清晰说明事实、复现步骤和技术依据。
- 对代码和设计提出具体、可执行的反馈。
- 尊重维护者和贡献者的时间。
- 在讨论安全、隐私和数据问题时保持谨慎。

## Unacceptable Behavior

- 人身攻击、骚扰、歧视性言论或威胁。
- 公开泄露他人的私密信息、API Key、题库数据或业务数据。
- 在 issue、PR 或讨论中持续发布无关内容。
- 故意误导项目使用者或隐瞒高风险行为。

## Enforcement

维护者可以对违反本准则的内容进行编辑、隐藏或删除，并在必要时限制相关账号继续参与项目讨论。
